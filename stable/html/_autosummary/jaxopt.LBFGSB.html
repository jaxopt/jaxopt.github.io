L-BFGS-B solver.

L-BFGS-B is a version of L-BFGS that incorporates box constraints on variables.

Type

value_and_grad

whether fun just returns the value (False) or both the value and gradient (True). See base.make_funs_with_aux for details.

Type

Union[bool, Callable]

has_aux

whether fun outputs auxiliary data or not. If has_aux is False, fun is expected to be scalar-valued. If has_aux is True, then we have one of the following two cases. If value_and_grad is False, the output should be value, aux = fun(...). If value_and_grad == True, the output should be (value, aux), grad = fun(...). See base.make_funs_with_aux for details.

Type

bool

Type

tol

tolerance of the stopping criterion.

Type

float

stepsize

a stepsize to use (if <= 0, use backtracking line search), or a callable specifying the positive stepsize to use at each iteration.

Type

Union[float, Callable[[Any], float]]

linesearch

the type of line search to use: “backtracking” for backtracking line search or “zoom” for zoom line search or “hager-zhang” for Hager-Zhang line search.

Type

str
