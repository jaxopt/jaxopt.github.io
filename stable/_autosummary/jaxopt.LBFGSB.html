L-BFGS-B solver.

L-BFGS-B is a version of L-BFGS that incorporates box constraints on variables.

has_auxÔÉÅ

whether fun outputs auxiliary data or not. If has_aux is False, fun is expected to be scalar-valued. If has_aux is True, then we have one of the following two cases. If value_and_grad is False, the output should be value, aux = fun(...). If value_and_grad == True, the output should be (value, aux), grad = fun(...). See base.make_funs_with_aux for details.
