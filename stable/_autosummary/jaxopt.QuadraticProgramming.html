jaxopt.QuadraticProgramming¶

class jaxopt.QuadraticProgramming(matvec_Q=None, matvec_A=None, maxiter=1000, tol=1e-05, implicit_diff_solve=None)[source]¶

Quadratic programming solver.

The objective function is:

0.5 * x^T Q x + c^T x subject to Gx <= h, Ax = b.

run(init_params=None, params_obj=None, params_eq=None, params_ineq=None)[source]¶

Runs the quadratic programming solver in CVXPY.

The returned params contains both the primal and dual solutions.

Parameters

  • init_params (Optional[Tuple]) – ignored.

  • params_obj (Optional[Tuple[ndarray, ndarray]]) – (Q, c) or (params_Q, c) if matvec_Q is provided.

  • params_eq (Optional[Tuple[ndarray, ndarray]]) – (A, b) or (params_A, b) if matvec_A is provided.

  • params_ineq (Optional[Tuple[ndarray, ndarray]]) – = (G, h) or None if no inequality constraints.

Return type

OptStep

Returns

(params, state), params = (primal_var, dual_var_eq, dual_var_ineq)
