implicit_diff_solve(b)¶

Solves A x = b using jax.lax.solve.

This solver is based on an LU decomposition. It will materialize the matrix A in memory.

Parameters

  • matvec (Callable) – product between A and a vector.

  • b (ndarray) – array.
